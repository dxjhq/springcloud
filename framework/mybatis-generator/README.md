
# mybatis代码生成器中文定制特别版

## 版本

````
3.1.0.1
````

## 介绍

````
1.基于MyBatis Generator 1.3.5源码改动的特别定制版
2.支持Lombok表达式
3.支持中文注释
4.增加了selectSelective动态查询方法
5.增加了Dao处理层
6.默认屏蔽了insert,updateByPrimaryKey等非动态SQL的处理
````

## 使用方法
````    
1.配置generatorConfig.properties
2.配置generatorConfig.xml
3.运行Run.java
4.静静的看着奇迹的发生
````
